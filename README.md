## 📝 Project Overview
**Doc-Genie** is an intelligent Python documentation assistant developed as part of the **Naan Mudhalvan (SmartBridge)** initiative. The platform leverages **Abstract Syntax Tree (AST)** analysis to automatically generate professional docstrings (Google/NumPy styles) by analyzing function signatures, parameters, and internal logic.

## 🚀 Key Features
- **Automated AST Analysis**: Extracts function names and arguments directly from source code.
- **Deep Logic Detection**: Identifies loops (for/while) and conditional branching (if-statements).
- **Interactive Web UI**: Powered by **Gradio** for a seamless user experience in the browser.
- **Professional Exporting**: Generates and downloads documentation as **PDF reports** using ReportLab.
- **Style Flexibility**: Supports industry-standard documentation formats.

## 🛠️ Tech Stack
- **Language:** Python 3.8+
- **Analysis Engine:** `ast` (Abstract Syntax Tree)
- **UI Framework:** Gradio
- **PDF Generation:** ReportLab
- **Platform:** Google Colab & GitHub

## 📂 Project Structure
- `Doc_Genie_Main.ipynb`: The primary notebook containing the analysis logic and Gradio interface.
- `README.md`: Project documentation and setup instructions.
- `requirements.txt`: List of Python dependencies.

## 📖 How to Run the Project
1. Open the repository on GitHub.
2. Click the **"Open in Colab"** badge at the top of this file.
3. In Google Colab, go to **Runtime > Run all**.
4. Scroll down to the Gradio output, paste a Python function, and click **"Generate & Export"**.
5. Download the generated PDF report for your records.
